Artworkc back to ToC or Class ToC

IRI: http://w3id.org/arto#Artwork

Represents a created object intended to be aesthetically pleasing or convey artistic meaning. This class encompasses various art forms including paintings, sculptures, photographs, and digital art. Unlike CIDOC-CRM's E22_Human-Made_Object which covers all manufactured items, arto:Artwork specifically focuses on objects created with artistic intent. The class inherits essential properties from schema:CreativeWork (creator, dateCreated, description) while adding art-specific constraints such as mandatory visual representation (edm:isShownBy) and current location tracking. Each artwork must contain at least one scene and have defined dimensions, supporting both traditional and contemporary art analysis.
